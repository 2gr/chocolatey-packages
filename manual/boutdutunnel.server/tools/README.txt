BoutDuTunnel requires:

- the .NET Framework v2.0 or later (http://www.microsoft.com/downloads)
  Best choice if you are behind a NTLM proxy (ISA Server) and/or running windows
 
- OR MONO v1.9.1 or later (http://www.mono-project.com)
  Best choice if you support open source software and/or running UNIX/Linux/Mac OS X/Solaris
